Migration de ce blog

Ce blog a longtemps été propulsé par le moteur de blog Dotclear, un moteur de blog écrit en PHP développé par des français (Olivier Meunier au départ, Franck Paul désormais). Il est désormais propulsé par Hugo. J’ai beaucoup utilisé Dotclear, je l’ai beaucoup aimé. A l’époque l’organisation du code PHP de Dotclear était bien meilleure que celle de Wordpress (j’avoue n’avoir pas regardé le code de Wordpress aujourd’hui).
