So funky...

avril 19, 2009 · franek

Si vous cherchez à vous déguiser selon le style sixties/seventies (ou pas), je vous conseille le magasin Tim Bargeot. Perso, je suis ressorti avec une veste du plus bel effet.

Un détail, le magasin se trouve à Paris près des Halles, 3 rue de Turbigo.
