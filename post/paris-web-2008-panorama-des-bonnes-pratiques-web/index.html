Paris Web 2008 - panorama des bonnes pratiques Web

December 20, 2008 · franek

J’ai assisté à Paris Web 2008 (je vous ai dit que c’était très bien ?) mais je n’ai pas pu participer à toutes les conférences (notamment les conférences décideurs). Les vidéos viennent d'être publiées.

Je commence par celle de François Nonnenmacher sur les bonnes pratiques Web. Je la recommande vivement à toute personne en charge de projets webs.
